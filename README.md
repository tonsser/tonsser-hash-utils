# Tonsser Hash Utils

A collection of classes for dealing with hashes

## Installation

Add this line to your application's Gemfile:

```ruby
gem "tonsser_hash_utils", "~> 1.0"
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install tonsser_hash_utils
